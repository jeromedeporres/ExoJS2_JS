HTML, CSS & JavaScript

Code HTML

Le langage HTML, HyperText Markup Langage, a été conçu au début des années 1990, par Tim Berners-Lee, le créateur du Web. La première version du HTML fut réalisée en 1993.

Code CSS

Le langage CSS, Cascading Style Sheet, a été conçu quelques années après l’apparition du Web, afin de séparer le contenu du style. La première version du CSS fut réalisée en 1997.

Code JavaScript

Le langage JavaScript, a été conçu avant le CSS, afin d'ajouter un aspect interractif aux pages-web. La première version du JavaScript fut réalisée en 1995.
